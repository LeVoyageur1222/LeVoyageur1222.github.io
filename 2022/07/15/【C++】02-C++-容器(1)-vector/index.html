【C++】02 C++ 容器(1) vector

发表于 2022-07-15 分类于 C++

本文将介绍C++ STL中常见的动态列表容器vector的基本用法，包含创建、增删、访问、遍历、排序等方面内容，以对C++其他容器类型进行启发。

vector简介

vector是封装动态大小数组的顺序容器，可以存放各种类型数据的对象。此处：

  • 动态大小：和C内置数组不同，容器在创建之后，可以动态往其中增删对象；
  • 顺序：指容器中的对象按照线性顺序排序，可以通过下标访问。
